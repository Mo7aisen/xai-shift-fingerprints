# Availability Statements (Gate-6)

## Data Availability
Code, analysis scripts, audit reports, and the Gate-6 reproducibility package are available at `https://github.com/Mo7aisen/xai-shift-fingerprints`. The reproducibility bundle is distributed as `GATE6_REPRO_BUNDLE_gate6_final_2026-02-17.tar.gz` (SHA256: `22b185a2a4c21afe5fd3ced691b06acd3cb194b5ba60e0b6c8c1e9b439597ec5`).

## Code Availability
All source code required to reproduce Gates 1-6 (configurations, scripts, tests, and governance manifests) is provided in this repository. A long-term DOI archive entry (Zenodo/Figshare) can be linked once minted.
